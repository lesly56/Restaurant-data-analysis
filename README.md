# Restaurant-data-analysis
Data analysis project focused on analyzing restaurant sales, customer behavior and business performance using Excel, SQL, Python and Power BI

## Business Questions
This analysis aims to answer questions such as:

- Which products generate the most revenue?
- What are the best-performing sales periods?
- Which categories have the highest demand?
- How do customer purchasing patterns change over time?
- What opportunities exist to improve business performance?

## Tools & Technologies

- Excel (Data cleaning, Pivot Tables, Dashboard)
- SQL (Data querying and analysis)
- Python (Pandas, Matplotlib, Seaborn)
- Power BI (Interactive dashboard)

## Dataset

This project uses a synthetic restaurant sales dataset generated with Python.

The dataset contains 20,000 restaurant transactions, including:

- Order information
- Customer information
- Product details
- Sales revenue and profit
- Discounts
- Payment methods
- Restaurant locations
- Dates and order times

The dataset was created for data analysis practice and business insights generation.

## Data Analysis Process

### 1. Data Cleaning
- Removed duplicates
- Handled missing values
- Standardized data formats

### 2. Exploratory Data Analysis
- Analyzed sales trends
- Identified top-selling products
- Compared revenue across categories

### 3. Data Visualization
- Created dashboards and visual reports to communicate insights.
